Al ser cadenas de 8 caracteres binarios, es decir, un byte completo. El valor máximo en binario de 8 bits es: 255 en decimal.
- a = 11111111
- b = 11111111

Esto equivale a 255 en decimal para ambos.
255 × 255 = 65,025
